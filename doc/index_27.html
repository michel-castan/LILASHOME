- 5.2.1 -

Panel

The Panel element has a "layout" attribute for specifying content organization. Possible layout values ​​are given in the following subsections. The element may have the "name" attribute to identify it. The parent element "awt" is in fact identical to an item "Panel" and can have the same "layout" and "name" attributes.

An empty element with name "instance" can also be used to reference an element defined by a awt defined element in the specified module by the "module" attribute (the "module" attribute can also indicate "this" if the element is defined internally within the module).

<awt colonnes="1" layout="grid" name="awt6"> <instance awt="awt3" module="this" name="instaw3"/> <instance awt="awt3" name="instaw3bis"/> </awt>

5.2.1.1 none

no layout (default)

Example :

<Panel> <Text signal="x" taille="10" editable="true"/> <Display signal="y"/> </Panel>

5.2.1.2 grid

screen grid. Two attributes are used: rows and columns to indicate respectively the number of rows or columns (this number can be equal to 0 for line or column but not both).

Example :

<Panel layout="grid" colonnes="1"> <Text signal="x" taille="10" editable="true"/> <Display signal="y"/> </Panel>

5.2.1.3 border

the content indicates the position of each element using the tags page_start, page_end, line_start, line_end, center.

Example :

<Panel layout="border"> <page_start> <Text signal="x" taille="10" editable="true"/> <Display signal="y"/> </page_start> <center> <ImageSVG href="tst.awt.sqrt"/> </center> </Panel>

5.2.1.4 tab

for an organization according to a list of tabs. Each element is specifying the title tab.

Example :

<Panel layout="tab"> <onglet titre="onglet 1"> <Text signal="x" taille="10" editable="true"/> <Display signal="y"/> </onglet> <onglet titre="onglet 2"> <ImageSVG href="tst.awt.sqrt"/> </onglet> </Panel>

It is possible to associate a Tab type of panel, an integer type signal to know the active panel number. The details of this signal is via the attribute "signalSelectedIndex" :

<signal mode="dummyIntern" name="tabSelectedIndex" type="java.lang.Integer"/> <Panel layout="tab" signalSelectedIndex="tabSelectedIndex"> <onglet titre="onglet 1"> <Text signal="x" taille="10" editable="true"/> <Display signal="y"/> </onglet> <onglet titre="onglet 2"> <ImageSVG href="tst.awt.sqrt"/> </onglet> </Panel>

5.2.1.5 scroll

The element will be positioned in a panel having, if necessary, positioning lifts.

Example :

<Panel layout="scroll"> <ImageSVG href="tst.awt.sqrt"/> </Panel>

5.2.1.6 flow

The items will be organized with a flow layout. It is possible to indicate the alignment of the elements through the optional attribute "align" whose possible values ​​are CENTER, LEADING, LEFT, RIGHT or TRAILING (see description in the java.awt.FlowLayout class).

Example :

<Panel layout="flow"> <ImageSVG href="tst.awt.sqrt"/> </Panel> <Panel layout="flow" align="CENTER"> <ImageSVG href="tst.awt.sqrt"/> </Panel>
