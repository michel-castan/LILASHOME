- 2.3.1 -

demo.ctrud.CtrUD_tst

Then, we create first a test module for demo.ctrud.CtrUD. We use the command :

$ lilas /demo/ctrud/CtrUD.lilas -to=tst File test demo/ctrud/CtrUD_tst.lilas created !

Now we have to fix a value for constant signal size (for example 32) and provide a script sequence :

CtrUD_tst.lilas :
<?xml version="1.0" encoding="UTF-8"?> <module> <signal init="32" mode="constant" name="size" type="java.lang.Integer"/> <signal init="0b0" name="clkCircuit" type="lilas.type.StdLogic"/> <signal init="0b0" name="clear" type="lilas.type.StdLogic"/> <signal init="0b0" name="upDown" type="lilas.type.StdLogic"/> <signal init="0bU:size" name="ctr" type="lilas.type.StdLogic"/> <CtrUD clear="clear" clkCircuit="clkCircuit" ctr="ctr" name="instCtrUD" size="size" upDown="upDown"/> ... <script name="clkPeriod"> <waitStable/> <next signal="clkCircuit" value="0b1"/> <waitStable/> <next signal="clkCircuit" value="0b0"/> <waitStable/> </script> <script name="main"> <group> <next signal="clkCircuit" value="0b0"/> <next signal="clear" value="0b1"/> <next signal="upDown" value="0b0"/> </group> <clkPeriod/> <check signal="ctr" value="0x0000_0000"/> <next signal="clear" value="0b0"/> <clkPeriod/> <check signal="ctr" value="0x0000_0001"/> <clkPeriod/> <check signal="ctr" value="0x0000_0002"/> <next signal="upDown" value="0b1"/> <clkPeriod/> <check signal="ctr" value="0x0000_0001"/> <clkPeriod/> <check signal="ctr" value="0x0000_0000"/> <clkPeriod/> <check signal="ctr" value="0xFFFF_FFFF"/> <next signal="clear" value="0b1"/> <waitStable/> <check signal="ctr" value="0x0000_0000"/> <print>Done!</print> </script> </module>
The execution of this script give :
$ lilas /demo/ctrud/CtrUD_tst.lilas -awt=awt Lilas ready ! Done! **Script main done (7 validated checks of 7) at 9.492ms
Note: The green vertical lines that appear on the history of ctr signal indicate a check success (in the case of failure, the line appears in red).
