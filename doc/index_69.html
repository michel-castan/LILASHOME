- 6.1.3 -

Aff7Seg

This module models a 7-segment display:

Aff7Seg.lilas :
<?xml version="1.0" encoding="UTF-8"?> <module> <signal init="0x00" mode="dummyExtern" name="digit" sensitive="true" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="a" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="b" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="c" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="d" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="e" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="f" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="g" type="lilas.type.StdLogic"/> <signal init="0b0" mode="internal" name="p" type="lilas.type.StdLogic"/> <awt name="awt"> <ImageSVG img="lilas.bib.awt.Aff7Seg"> <ElementSVG_StdLogicSpecial_Inv idRef="a" signal="a"/> <ElementSVG_StdLogicSpecial_Inv idRef="b" signal="b"/> <ElementSVG_StdLogicSpecial_Inv idRef="c" signal="c"/> <ElementSVG_StdLogicSpecial_Inv idRef="d" signal="d"/> <ElementSVG_StdLogicSpecial_Inv idRef="e" signal="e"/> <ElementSVG_StdLogicSpecial_Inv idRef="f" signal="f"/> <ElementSVG_StdLogicSpecial_Inv idRef="g" signal="g"/> <ElementSVG_StdLogicSpecial_Inv idRef="p" signal="p"/> </ImageSVG> </awt> </module>

To control this module, we can use the encoding module:

Aff7SegEncoder.lilas :
<?xml version="1.0" encoding="UTF-8"?> <module> <doc>Pour coder une valeur sur 4 bits et fournir le code 8 bits correspondant pour les afficheurs 7 segments</doc> <signal init="0x0" mode="in" name="val" sensitive="true" type="lilas.type.StdLogic"/> <signal init="0x00" mode="out" name="digit" type="lilas.type.StdLogic"/> <code language="logic"><![CDATA[ digit <= with val select ( 0b01110001, 0b01100001, 0b10000101, 0b01100011, 0b11000001, 0b00010001, 0b00001001, 0b00000001, 0b00011111, 0b01000001, 0b01001001, 0b10011001, 0b00001101, 0b00100101, 0b10011111, 0b00000011); ]]> </code> </module>
