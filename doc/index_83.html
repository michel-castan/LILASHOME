GenClockParametrableClkInv

The module GenClockParametrableClkInv is used to generate a clock signal clk on the falling edge of clk100M pulse lasting one clock period over n periods of the basic clock (after the reset, the pulse start on the first falling edge of the clock):

  • n = 0 (snapshot)
  • n = 1 (snapshot)
  • n = 2 (snapshot)
  • n = 3 (snapshot)

In step by step mode, signal clk is triggered n + 1 periods after the start signal. No minimum periods are expected before a new trigger (provided that in the meantime, the start signal is dropped to zero).

  • n = 0 (snapshot)
  • n = 1 (snapshot)
  • n = 2 (snapshot)
