EtageP1

Le traitement réalisé sur les données utilise les valeurs des tableaux cos et sin :

cos = new double[taille/2]; sin = new double[taille/2]; for(int i=0; i<taille/2; i++) { cos[i] = Math.cos(-2*Math.PI*i/taille); sin[i] = Math.sin(-2*Math.PI*i/taille); }
Ces deux tableaux peuvent être réduits à un seul de taille réduite de moitié (si des problèmes de tailles venaient apparaître) :
tab = new double[taille/4+1] for(int i=0; i<=taille/4; i++) { tab[i] = Math.sin(2*Math.PI*i/taille); } avec i<=taille/4 i>taille/4 sin[i] = -tab[i] -tab[taille/2 -i] cos[i] = tab[taille/4-i] -tab[i-taille/4]
Remarque : en utilisant tab réduit, on restreint les calculs du sinus et du cosinus entre 0 et PI/4.
