6.2.2.1 lilas.bib.logic.flipflop.D

LILAS description : D.lilas

Signals :

  • clk : (in) clock signal (1 bit)
  • dIn : (in) input data (1 bit)
  • dOut : (out) output data (1 bit)

The flip-flop stores the information on the rising edge of the clock. The stored information is available on DOUT.

Launch of the test:

$ lilas lilas/bib/logic/flipflop/D_tst.lilas -awt=awt -script=test Système prêt! Script lancé 1 : L'entrée Din bouge, la sortie reste stable 2 : Sur le front montant de l'horloge, l'entrée est mémorisée 3 : L'entrée Din bouge, la sortie reste stable 4 : L'horloge redescend, la sortie ne bouge pas 5 : On remet la bascule à 0 6 : fin du test

6.2.2.2 lilas.bib.logic.flipflop.D

LILAS description : DEnAC.lilas

Signals :

  • clk : (in) clock signal (1 bit)
  • en : (in) enable signal (1 bit)
  • clr : (in) clear signal (1 bit)
  • dIn : (in) input data (1 bit)
  • dOut : (out) output data (1 bit)

The flip-flop stores he information on the amount of the clock edge only if the signal en is equal to 1. The stored information is available on dOut. The signal clrr allows a reset of flip flop asynchronously.

Launch of the test:

$ lilas lilas/bib/logic/flipflop/DEnAC_tst.lilas -awt=awt -script=test Système prêt! Script lancé 1 : L'entrée Din bouge, la sortie reste stable 2 : En n'est pas actif, l'entrée n'est pas mémorisée 3 : En est actif, l'entrée est mémorisée sur le front montant de l'horloge 4 : L'entrée Din bouge, la sortie reste stable 5 : L'horloge redescend, la sortie ne bouge pas 6 : On remet la bascule à 0 7 : fin du test **Script test terminé (13 checks validés sur 13) à 22.286ms
