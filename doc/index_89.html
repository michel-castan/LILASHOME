FPGA board

LILAS considers three FPGA boards:

  • board Nexys3 from Digilent
  • boards minispartan6 from Scarab Hardware (LX9 et LX25)
Names recognized by LILAC for these target cards are :
  • nexys3 (Board Nexys3)
  • ms6lx9 (Board minispartan6 with Spartan 6 LX9)
  • ms6lx25 (Board minispartan6 with Spartan 6 LX25)

The indication of the target board can be done in several ways. By decreasing priority, we can specify the target:

  • at the definition of a module through the board attribute:
    <module board="nomDeLaCible"> ... </module>
  • at the lilas command with the option -board:
    $ lilas ... -board=boardName ...
  • in the configuration file with var element:
    <config> ... <var name="board" value="boardName"/> ... </config>
