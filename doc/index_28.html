ListDisplay

The ListDisplay element used to display a series of signals or modules organized in a tree structure (the signals and modules are leaves, List elements are the nodes). Visualization is a JTreeTable (ie a table whose first column is organized in a tree structure (JTree). The other three columns represent the type of signal, the current value and history. It is possible to impose a title for each of the three columns through titleCol1, titleCol2 and titleCol3 attributes (these attributes are optional). It is possible to specify a size for each column with the sizeCol0, sizeCol1, sizeCol2 and sizeCol3 attributes (these attributes are optional). An equal size 0 can not view the corresponding column (sizeCol0 must always be different from 0). As any element awt, optional name attribute is used to impose a particular name. It is also possible to set the duration one pixel for the history display, via the optional attribute "duréePixel" (default duréePixel = 2000000).

.

The ListDisplay element can contain the following elements:

  • node: to view a signal (indicated by the attribute 'signal') or a module (indicated by the attribute 'module') and a particular signal system (indicated by the attribute 'system', the allowable values ​​are system="Date" for the date change history, system="Activity" for the overall system activity, system="Messages" for messages edited with a color other than black). In the case of a signal, the attribute 'editable' allow to changes the signal value by clicking in the cell in column 2 (numbering from 0), the attribute 'text' gives a text to replace the full name signal in column 1.
  • list : To view a sub-tree. Such an element have similar contents as ListDisplay element. Attribute 'open' indicate whether, during the creation of the table and if the element is visible, the subtree of the content should be displayed or not.

Example :

<awt name="panneau"> <ListDisplay titleCol3="La liste pan8" titleCol1="col1" titleCol2="col2" duréePixel="100000L"> <node system="Date"/> <node system="Activité"/> <node signal="x" editable="true"/> <node signal="y"/> <node module="this"/> <list titleCol3="le premier titre supplémentaire" open="true"> <node signal="y"/> <node signal="x"/> </list> <list titleCol3="le deuxième titre supplémentaire"> <node signal="y"/> <node signal="x" editable="true"/> </list> </ListDisplay> </awt>

It is possible to position brands (vertical magenta line) by clicking right mouse button on any position.
