MemoryDisplay

Element used to display the contents of an instance of the class lilas.base.Memory whose name the associated variable is provided by the attribute "memory".

Example :

<MemoryDisplay memory="memPrime"/>

The MemoryDisplay element may use the following attributes (none or both) to impose a size in each column: sizeCol0 and sizecol1.

Few examples :

  • <awt colonnes="1" width="200" height="200" layout="grid" name="mempan1"> <MemoryDisplay memory="mem"/> </awt>
  • <awt name="mempan2" width="200" height="200" colonnes="1" layout="grid"> <MemoryDisplay memory="mem" sizeCol0="50" sizeCol1="100"/> </awt>
  • <awt colonnes="1" width="200" height="200" layout="grid" name="mempan3"> <MemoryDisplay memory="mem" model="ModeleMemoire"/> </awt>
  • <awt colonnes="1" width="200" height="200" layout="grid" name="mempan4"> <MemoryDisplay memory="mem" sizeCol0="50" sizeCol1="100" model="ModeleMemoire"/> </awt>
