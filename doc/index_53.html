load

The element instruction "load" is used to initialize a memory from a initialization file (extension ".mem"). For example, the following statement initializes the memory element "Test1.mem" From File "tst/mem/MemInit.mem" available in the program launch classpath.

<load memory="Test1.mem" memInit="tst.mem.MemInit"/>
