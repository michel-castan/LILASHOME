Installation

The complete application LILAS reside in LILASHOME directory which can be downloaded from the GitHub site :

$ git clone https://github.com/michel-castan/LILASHOME
This command will create a directory named LILASHOME. This directory can be anywhere in your file system. The inclused file LILASHOME/lib/lilasV4.jar is en executable JAR file. The typical invocation is java -jar LILASHOME/lib/lilasV4.jar from a command line or by clicking on this file into a file browser window.

The next step is to create a working directory for your project in which we will define your lilas modules. This working directory can be anywhere in your file system with any name; however it is better to create this directory outside LILASHOME directory (thus, we can create a git repository different of LILASHOME one). Let's name LilasProject this directory.

The final step will be to create a configuration file for LILAS. Again, this file can have any name and can be anywhere in your file system. However, if we call it lilasConfig.xml and if it reside at the root of the working directory LilasProject, this will simplify the run command for lilas. The simplest way to create this configuration file is to launch the executable JAR archive LILASHOME/lib/lilasV4.jar either by double clicking on the name of this archive from the file manager, or by the command java -jar lilasV4.jar. By clicking on the button Save, the file lilasConfig.xml will be created in the directory LilasProject, and also a script file for launching LILAS (lilas.sh or lilas.bat depending on the operating system).
