Text

Element for displaying the form of a string (JTextField) the content of a signal. The size attribute defines the size (number of columns: see javax.swing.JTextField) of the JTextField component. The editable attribute to specify whether the widget is editable or not (default) interactively. The optional label attribute prefix of the area given by an instance of the class javax.swing.JLabel.

Example :

<Text signal="x" taille="10" editable="true"/> <Text signal="x" taille="10" label="x :" editable="true"/>
