C (experimental)

We can define a module with a C syntax. However, only ths subset of C common to Java can be use for now (the Java traduction is a simple text copy).

The activation description, the initialisation description are given with a function description withour parameter. All others definition (var or procedure) are definition of field or procedure for the module :

<module> <signal mode="in" name="no" sensitive="true" type="java.lang.Integer" init="-1"/> <signal mode="in" name="dIn" type="java.lang.Double" init="0.0"/> <signal mode="out" name="k" type="java.lang.Integer"/> <signal mode="out" name="x" type="java.lang.Double"/> <signal mode="out" name="y" type="java.lang.Double"/> <code language="C"> void initialisation() { } void activation() { k = no; x = dIn; y = 0.0; } </code> </module>
