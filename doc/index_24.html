lilas.base.Port

The port is the interface between the signal and the module. This class contains the associated elements for communication between modules :

  • the sensitivity of the signal for a module (the sensitivity define activation of module following the presence of a value on the associated signal).
  • the mode defining the direction of data exchange through the signal. The different modes are : IN, OUT, INOUT, CONSTANT, DUMMYINTERN, DUMMYEXTERN.
  • the propagation of the data in the output signal
