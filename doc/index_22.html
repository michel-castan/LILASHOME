- 5.1 -

lilas.base

This section presents the essence of LILAS base classes.

The package lilas.base contains the basic classes (abstract in general). These classes are the basic building blocks for different applications, from RTL modeling, to multi-core programming.

Class lilas.base.Module shows a basic element (computing unit, VHDL entity, ...) or a subset of base members and/or module. A module thus represents the logical organization of a system or circuit. The exchange of information between modules is done through instances of the class lilas.base.Signal. Each module transmits data signals via the instances of the class lilas.base.Port.

Class lilas.base.SystemL represents the location of a set of modules on a particular structure. For a VHDL circuit, an instance of lilas.base.SystemL represent an FPGA.

The example demo.sqrt.SQRT from the previous section Quick start tutorial shows how these classes are used to build a LILAS appplication.
