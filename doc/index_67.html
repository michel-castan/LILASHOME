ElementSVG_StdLogic

Java subclass of lilas.awt.ElementSVG<StdLogic>. It allows to associate the svg element the following actions:

  • change the value with the logical-not of the current value of a click
  • offer a choice of menu logic values ​​1 bit with a click on the active shift key
  • change the text and color on a change of value on the associated signal
