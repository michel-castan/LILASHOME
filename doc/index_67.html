Forward button

This module provides a button to advance with a given period of the clock when the system is stable. Two files describe this module:

  • BoutonAvance.lilas
  • BoutonAvance.java

To use this button, just add a module instance in a lilac module and to refer to it throughout awt defined in BoutonAvance:

<module> ... <lilas.bib.awt.BoutonAvance name="btn" delaiInit="2000"/> ... <awt layout="border" name="awt"> ... <page_end> <instance awt="btn.awt"/> </page_end> </awt> </module>
