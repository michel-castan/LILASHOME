  • MemoireROM :

    MemoireROM.lilas :
    <?xml version="1.0" encoding="UTF-8"?> <module> <doc>Mémoire ROM simple</doc> <signal mode="constant" name="sizeWord" type="java.lang.Integer"/> <signal mode="constant" name="sizeAdr" type="java.lang.Integer"/> <signal mode="constant" name="sizeMem" type="java.lang.Integer" init="256"/> <signal mode="in" name="adr" type="lilas.type.StdLogic" sensitive="true" size="sizeAdr"/> <signal mode="out" name="dOut" type="lilas.type.StdLogic" size="sizeWord"/> <signal mode="internal" name="mem" type="lilas.type.ArrayStdLogic" init="256:0xUUUU"/> <code language="logic" duréeActivation="1ns"><![CDATA[ //array mem <= ( 0xFF:8, 4:4 => 0xFE, 1:4=>0xFD, 0xFC, 0xFB, 0b0101 => 0xFA, 0xF9, 0xF8, 0xF7, 0xF6, 0xF5, 0xF4, 0xF3, 0xF2, 0xF1, 0xF0); dOut <= mem(adr); ]]></code> </module>

    It is possible to initialize the memory directly in the statement of the signal, using the attribute "memInit" (and possibly the "assembler" attribute):

    <signal mode="internal" name="mem" type="lilas.type.ArrayStdLogic" init="256:0xUUUU" memInit="tst.code.logic.memoire.Rom"/> --> initialise with the file content of tst/code/logic/memoire/Rom.mem or the file content of tst/code/logic/memoire/Rom.ass if attribute 'assembleur' is given
