Nexys3

Module Nexys3.lilas manages the various input/output elements available on the Nexys card. It also generates the enable signal to the clock from the clock 100M map. It uses two modes: debug mode and IO mode. The IO mode allows to use the devices in the nexys3 card as input/output device of the logical system contained in the FPGA. The debug mode can display debug information (eg when in step mode to view all communications over to DSTM module).

Different module interface signals are:

  • Logical system side:
    • Inputs
      • ledDebug : values ​​to be displayed on LED in debug mode
      • ledIO : values ​​to be displayed on LED in IO Mode
      • digitDebug : values ​​(hex mode) to display the debug mode displays
      • digitIO : values ​​(graphics mode) to display on the IO mode displays
    • Outputs
      • sw : SWITCH values, sampled at 190Hz (100Mhz/2^19)
      • btn : Button values, sampled at 190Hz (100Mhz/2^19)
      • reset : reset yhe logic circuit
      • clkNexys3 : clk parameterizable for the main logic circuit..
      • clkDebug : clk enable debug
  • DSTM side:
    • Inputs
      • softReset : reset command issued by the DSTM Module
      • clkDebugIn : clk debug signal sent by the DSTM
      • clkPasAPas : clk signal sent by the DSTM
    • Outputs
      • hardreset : signal obtained by simultaneously pressing and BTNL BTNR button. Causes the reset of the logic circuit and that of the DSTM
  • Side nexys3 card:
    • Inputs
      • ledDebug : values ​​to be displayed on LED in debug mode
      • ledIO : values ​​to be displayed on LED in IO Mode
      • digitDebug : values ​​(hex mode) to display the debug mode displays
      • digitIO : values ​​(graphics mode) to display on the IO mode displays
      • swBroches : connections on pins from the switches
      • btnBroches : connections on pins from buttons
      • clk100M : 100Mhz clock
    • Outputs
      • ledsBroches : connections on pins to LEDs
      • Seg : connections to the displays segments
      • AN : selection of connections of the display

Clock operation: The Nexys3 circuit allows to generate 2 clocks and called clkNexys3 and clkDebug from the 4-bit control word: cmdGenClk (4 significant bits sent via the USB cable).
