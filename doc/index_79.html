Nexys3IO

Module Nexys3IO.lilas manages the various input/output elements available on the Nexys card. It uses two modes: debug mode and IO mode. The IO mode allows to use the devices in the nexys3 card as input/output device of the logical system contained in the FPGA. The debug mode can display debug information.

Different module interface signals are:

  • Inputs
    • clk100M : 100Mhz clock
    • ledDebug : values ​​to be displayed on LED in debug mode
    • ledIO : values ​​to be displayed on LED in IO Mode
    • digitDebug : values ​​(hex mode) to display the debug mode displays
    • digitIO : values ​​(graphics mode) to display on the IO mode displays
    • softReset : reset request from jtag command
  • Outputs
    • swIO : SWITCH values, sampled at 190Hz (100Mhz/2^19)
    • btnIO : Button values, sampled at 190Hz (100Mhz/2^19)
    • hardReset : signal obtained by simultaneously pressing and BTNL BTNR button.
    • reset : HardReset or softReset
  • Pins Nexys3 FPGA
    • LedsPins
    • SwitchsPins
    • ButtonsPins
    • ANPins
    • SegPins
    • clk100M : 100Mhz clock
