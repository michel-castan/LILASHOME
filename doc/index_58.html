snapshot

The element instruction "snapshot" creates an SVG image of the current window in the module directory. The file name is given by the attribute "file".

<snapshot file="snapshot.svg"/>

Two optional parameters are used to select the visible time period:

  • début : time (length) from the beginning of the visible window; Default 0
  • fin : date (long) from the end of the visible window; current date by default.
<snapshot file="snapshot.svg" début="20000"/> <snapshot file="snapshot.svg" début="20000" fin="40000"/>

The date can also be given by a name (see the name-day association in the print order).

When a file name is not given, only fixed snapshot interval that is displayed in the window. The display is then frozen until the next change in the visible scale (eg with the mouse wheel).
