lilas.code

The element "code" is used to describe the execution of the activation of the module in a textual language. The attribute (required) 'language' indicates the grammar used for language. The item code only contains text that is translated into java to give directly the "activation ()" method of the module. In case of attribute language equals to "java", the element is empty; an associated java class give the activation method.

<code language="logic"> // description logique du demi-additionneur r <= a and b; s <= a xor b; </code>

The item "code" can use the optional attribute "duréeActivation" indicating the maximum time of activation of the module (integer or not followed by the indication of the unit of time used: sec, ms, microseconds or us, ns, ps (by default)). This period will give the execution time of the module when the system is run with the option -modeSimulatedTime.

<code language="logic" duréeActivation="2ns"> s <= a[3] xor b[3] xor ri[3]; </code>
