for

The element instruction "for" to repeat a sequence of instructions with full variable or StdLogic kind taking values ​​between "from" and "to". These are the values ​​"from" and "to" that determine the type of the loop variable. In the case of logical values, these values ​​must be converted into integer (ie not containing values ​​of 'Z', 'U', 'W', 'L', 'H' or '-').

<for var="varTypeInteger" from="valeurDébut" to="valeurFin"> <next signal="x" value="varTypeInteger"/> <print>nouveau x = <in signal="x"/></print> </for> <for var="varTypeStdLogic" from="0b00" to="0b11"> <next signal="y" value="varTypeStdLogic"/> <print>nouveau y = <in signal="y"/></print> </for>

The loop variable is only visible in the body of the loop.
